# DANISH Search Engine

Components:

1. Web Crawler
2. Indexing Engine
3. Ranking Engine
4. Knowledge Graph
5. Semantic Search

Flow:

Web Sources
      ↓
Crawler
      ↓
Indexer
      ↓
Vector Database
      ↓
Search API
      ↓
Frontend

Technologies:

Crawler:
- Scrapy

Indexing:
- Elasticsearch

Vector Search:
- Qdrant

Embeddings:
- Transformer Models

Ranking:
- Hybrid Search
